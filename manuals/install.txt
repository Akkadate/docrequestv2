การติดตั้งและเริ่มต้นใช้งาน

ก่อนอื่นให้สร้างโครงสร้างโฟลเดอร์ตามที่กำหนดไว้ด้านบน
วางไฟล์ตามตำแหน่งที่กำหนดไว้
สร้างฐานข้อมูลใน PostgreSQL และปรับแต่งไฟล์ .env ตามการเชื่อมต่อที่ถูกต้อง
เปิด Terminal และทำการติดตั้ง dependencies:

bashnpm install

สร้างโครงสร้างฐานข้อมูลโดยใช้ไฟล์ schema.sql:

bashpsql -U postgres -d document_request_system -a -f database/schema.sql

เริ่มต้นเซิร์ฟเวอร์:

bashnpm start
หรือเริ่มต้นในโหมดพัฒนา (มีการรีโหลดอัตโนมัติเมื่อมีการเปลี่ยนแปลงโค้ด):
bashnpm run dev

เข้าใช้งานระบบผ่าน URL: http://localhost:3000

หมายเหตุสำคัญ

ระบบนี้มีการรองรับ 3 ภาษา (ไทย, อังกฤษ, จีน)
มีระบบผู้ใช้ 2 ระดับ คือ นักศึกษาและผู้ดูแลระบบ
ผู้ดูแลระบบเริ่มต้น:

รหัสผู้ใช้: admin
รหัสผ่าน: admin123 (เข้ารหัสด้วย bcrypt ไว้ในฐานข้อมูล)


ระบบรองรับการอัปโหลดไฟล์หลักฐานการชำระเงิน
มีการแสดงผลแบบ Responsive สำหรับการแสดงผลบนอุปกรณ์ต่างๆ

โค้ดนี้เป็นเพียงส่วนหลักๆ ของระบบ อาจต้องเพิ่มฟีเจอร์อื่นๆ ตามความต้องการเพิ่มเติม เช่น การส่งอีเมลแจ้งเตือน การออกรายงานในรูปแบบ PDF เป็นต้น

ระบบนี้มีฟีเจอร์ที่ครบถ้วนตามความต้องการที่ระบุไว้ เช่น:

รองรับการใช้งาน 3 ภาษา (ไทย, อังกฤษ, จีน)
ระบบลงทะเบียนและเข้าสู่ระบบ
การขอเอกสารประเภทต่างๆ
การเลือกวิธีรับเอกสาร (รับเองหรือทางไปรษณีย์)
คำนวณค่าบริการอัตโนมัติ
อัปโหลดหลักฐานการชำระเงิน
ตรวจสอบสถานะคำขอเอกสาร
ระบบจัดการสำหรับผู้ดูแลระบบ
รายงานสำหรับผู้บริหาร
